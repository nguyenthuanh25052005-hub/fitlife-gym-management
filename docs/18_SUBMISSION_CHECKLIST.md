# Submission Checklist

- [ ] Link GitHub repository đã được chuẩn bị.
- [ ] README đã đầy đủ hướng dẫn chạy app, test, lint và demo.
- [ ] GitHub Actions CI chạy PASS.
- [ ] Coverage report có sẵn từ Jest.
- [ ] SPQM report đã cập nhật.
- [ ] Video demo <= 5 phút đã sẵn sàng.
- [ ] Hướng dẫn chạy app và demo account đã có.
- [ ] SonarQube dashboard screenshot nếu có.
- [ ] Tình trạng hoàn thành và phần còn lại đã ghi rõ.
