# Manual Deliverables

Các hạng mục sau cần được người dùng/nhóm thực hiện thủ công vì không thể tự tạo hoàn toàn trong repo:

- Screenshot SonarQube dashboard nếu có triển khai SonarQube/SonarCloud.
- Video demo <= 5 phút và upload lên Google Drive/YouTube.
- Add giảng viên làm collaborator trên GitHub.
- Screenshot GitHub Actions PASS sau khi pipeline chạy xanh.
- Báo cáo Word/PDF nếu giảng viên yêu cầu file riêng.
- Link GitHub repo cần copy vào README hoặc slide báo cáo.
